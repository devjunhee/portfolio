# 수정 내용

- Hero / About / Projects / Skills / Contact 구성으로 포트폴리오 전체 리디자인
- JIB 프로젝트를 Problem / Solution / Impact 중심으로 강화
- KAI 프로젝트에 DT, OT, FAT, SAT, 체계통합, 정적 분석, 신뢰성 검증 경험 반영
- 프로젝트 카드에 역할, 연계회사, 기술스택, 핵심 성과 표시
- 반응형 CSS 적용
- `npm run build` 성공 확인

## 적용 방법

1. 기존 `portfolio` 폴더 백업
2. 이 압축 파일을 풀고 기존 프로젝트 파일에 덮어쓰기
3. 터미널에서 실행

```bash
npm install
npm run dev
```

## 배포
